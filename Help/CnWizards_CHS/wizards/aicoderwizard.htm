AI 辅助编码专家

该专家能够连接并调用几个常用的 AI 公开引擎之一，提供 IDE 内的 AI 辅助编码功能。各 AI 引擎的 API 密钥需用户自行申请。使用前，请先运行该专家的配置，在对话框中填入申请的 API 密钥方能使用，如图所示：

AI 辅助编码专家设置对话框

该对话框列出了可用 AI 引擎的参数，可供用户选择使用的 AI 引擎，以及设置每一个 AI 引擎的模型、API 密钥等。

  • 当前 AI 引擎：选择当前要使用的 AI 引擎，该引擎一般必须配置有效的 API 密钥。

  • 超时秒数：网络通讯的超时秒数设置，0 表示使用系统默认值。

  • 使用代理：如勾选，则代表指定连接该 AI 引擎所需的代理服务器，可以是 HTTP 或 SOCKS5 代理，格式如“192.168.1.1:1080”。暂不支持用户名密码。

  • 对话时发送历史消息数：指定每次与 AI 引擎在聊天框内对话时携带最近几条历史记录以记忆上下文。

  • 收藏最大条目数：指定 AI 聊天窗口内收藏夹能容纳的最多条目数，0 表示无限制。

  • 编辑器中使用快捷键 (Ctrl +) Alt + Enter 续写代码：是否启用两个快捷键在编辑器中调用续写代码的功能，其中 Alt + Enter 是将 AI 撰写的代码直接输入至编辑器当前光标后，Ctr + Alt + Enter 则是将结果输出至 AI 对话窗口中供阅读、复制。

  • 请求地址：AI 引擎的 API 在线请求地址，由该 AI 引擎提供。

  • 模型名称：选择该 AI 引擎提供的可用的模型名称。

  • 在线获取模型列表：以当前页面参数在线调用该 AI 引擎提供商接口以在线获取最新的模型列表，如成功则更新在下拉列表中。注意部分 AI 引擎提供商不支持该操作。

  • 采样温度：即 Temperature 参数，控制 AI 思维的发散程度。

  • API 密钥：需填入在该 AI 引擎网站上注册并申请的合法 API 密钥。

  • 流式输出：不勾选则控制让 AI 引擎一次性返回完整应答数据，否则连续多次返回。

  • 申请：点击申请链接可在线注册并申请该 AI 引擎的 API 密钥。

  • 重置：该按钮可将当前 AI 引擎的设置恢复至默认状态，但不清空 API 密钥。

  • 快捷键：弹出快捷键设置对话框，设置菜单项的快捷键。

  • 确定：保存修改内容并关闭对话框。

  • 取消：取消修改并关闭对话框。

  • 帮助：显示帮助信息。

 

AI 辅助编码专家菜单

AI 辅助编码专家的菜单项如图所示：

  • 解释代码：向当前 AI 引擎发送请求，让其解释编辑器中选中的代码块，返回的内容将显示在 AI 对话窗口中。

  • 检视代码：向当前 AI 引擎发送请求，让其针对编辑器中选中的代码块进行检视，也就是常说的 Code Review。返回的检视意见将显示在 AI 对话窗口中。

  • 生成测试用例：向当前 AI 引擎发送请求，让其针对编辑器中选中的代码块生成一个无界面交互的测试用例。返回的结果将插入当前编辑器光标所在处，同时也显示在 AI 对话窗口中。

  • 续写代码：向当前 AI 引擎发送请求，让其针对编辑器中当前光标处的代码或注释续写一段代码，返回的结果将插入当前编辑器光标所在处。该功能默认热键为 Alt+Enter。

  • AI 对话窗口：显示 AI 对话窗口。

  • 设置：显示上文的设置对话框。

 

AI 对话窗口

AI 对话窗口如图所示。上部是聊天消息显示区，下部是本地聊天消息发送区：

  • 切换发送框：显示或隐藏发送消息的区域。

  • 清空消息：清空消息列表对话框。

  • 提问时附上编辑器内选中的代码：消息区内发送消息时，是否附上编辑器内选中的代码供 AI 分析。

  • 字体：修改消息框与发送框的字体。

  • 设置：显示上文的设置对话框。

  • 帮助：显示此帮助。

  • 收藏下拉菜单：显示收藏下拉菜单，供快速在本地聊天消息发送区输入收藏的内容。

  • 添加至收藏：将本地聊天消息发送区的内容添加至收藏菜单。

  • 清空收藏：删除全部收藏菜单。

  • 发送：发送本地聊天消息区的内容给当前 AI 引擎。

 

右键点击消息体将弹出上下文菜单：

  • 复制：复制选中的消息体内容。

  • 复制代码：复制选中的消息体中被 ``` 括起来的内容当作代码处理。

  • 添加到收藏：添加选中的消息体中的文字到收藏。
