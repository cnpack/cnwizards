已引用单元列表

显示当前单元所引用的单元名列表，可根据文件名查找过滤，可在搜索路径中搜索选中文件并打开。本功能支持 Pascal 和 C/C++ 源文件。对于 Pascal 源文件，本功能能列出它 uses 的文件名列表并显示是在 interface 部分还是 implementation 部分；对于 C/C++ 源文件，本功能可列出它 include 的文件名列表。

  • 打开所选单元：在 IDE 中打开所选择的单元，可以是多个单元。

  • 复制单元名：复制所选择的单元名称到剪贴板。

  • 选择所有单元：选中单元列表中的所有单元。

  • 取消选择：取消单元列表中的选择。

  • 反选单元：反向选择单元列表中的已选择单元。

  • 按单元名开头匹配：查找的时候，输入的字符从单元名的头部开始匹配，单元名头部含有输入字符的才能显示。

  • 按单元名的所有位置匹配：查找的时候，输入的字符匹配单元名的任何部分，单元名含有输入字符的才能显示。

  • 打开多个单元提示：打开多个单元时提示。

  • 字体：更改列表框字体。

  • 帮助：显示此帮助。

  • 关闭：关闭此窗口。
