IDE 界面汉化

除专家包自身多语言外，CnPack IDE 专家包还提供对部分 Delphi 版本 IDE 界面文本的汉化能力。该功能用于将 IDE 中部分英文菜单和界面文字替换为中文，便于中文用户日常使用。

该功能主要覆盖菜单与常见界面文本，因 Delphi 各版本差异较大，不同版本可覆盖范围可能不同。建议在启用后结合您的 IDE 版本进行实际确认。

启用条件

  • 当前专家包语言为中文。

  • 在 Languages 菜单中启用 IDE 界面汉化功能。

使用说明

  1. 在 Delphi IDE 中打开 CnPack -> Languages。

  2. 启用 IDE 界面汉化相关选项。

  3. 如有提示，请重启 IDE 以确保全部界面刷新。

如果您在特定版本中发现个别界面仍为英文，属于版本兼容差异的正常现象。可优先升级到较新版本的专家包以获得更完整的适配。
