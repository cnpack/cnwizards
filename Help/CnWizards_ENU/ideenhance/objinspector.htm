Object Inspector Enhancements Wizard

Object Inspector Enhancements Wizard provides a series of extended functionalities for the Object Inspector. Its configuration interface is shown in the figure below:

  • Enhance Property Editor Drawing: The feature adds bold display for non-default property values in the Object Inspector, similar to that in Delphi 7, and currently supports Delphi 5 and C++Builder 5. As shown in the figure below, after enabling this expert, in the Object Inspector of Delphi 5, non-default property values are displayed in bold, while component class properties are shown in brown.

  • Show Object Inspector Grid Lines: The Object Inspector in Delphi 6/7 features internal drawing with horizontal grid lines, whereas versions of Delphi 2005 and later do not include these lines. These two options allow for the individual settings to either draw or omit the grid lines, catering to the specific needs of the users.

  • Add "Comment Window" in Object Inspector's Popupmenu: Whether to add a "Comment Window" menu item in the Object Inspector's context menu. This menu item allows users to show or hide a comment window that targets the selected content in the Object Inspector. The comment window is designed for users who need to input notes for memory and reference. As shown in the figure below, the top part of the comment window dynamically displays the currently selected single component in the designer and its selected property in the Object Inspector, along with a corresponding comment edit box for editing note content. Additionally, the status bar at the bottom shows the class inheritance hierarchy of the currently selected component.

    • Type Name: Selected One component in current designer.

    • Type Comment: Enter Type comment for this component here.

    • Property/Event Name: Selected One property or event in Object Inspector.

    • Property/Event Comment: Enter property/event comment here.

    • Status Bar: The status bar at the bottom displays the class inheritance hierarchy of the currently selected component.

    • Clear: Clear all comments in this window.

    • Copy Name: Copy current type name, property name or event name.

    • Font: Change Display Font.

    • Toggle Grid Mode: Switch to a Property Grid mode to list all properties and events for a component to add batch comments.

    • Help: Show this help.

    •  

  • Change Object Inspector Font Size: If you think the font size of Object Inspector is too small, you can change its font size here.
