CnWizards Multi-Language Settings

CnPack IDE Wizards provides a built-in multi-language UI. You can switch the interface language in the Languages menu. After switching, CnWizards' menu items, dialogs and tips are updated immediately.

Currently, CnWizards supports seven languages:

  • Chinese Simplified

  • Chinese Traditional

  • English

  • German

  • French

  • Portuguese

  • Russian

It is recommended to use a language consistent with your team workflow, so option descriptions and prompt messages are easier to understand.

How to Use

  1. Open CnPack -> Languages in Delphi IDE.

  2. Select the target language.
