IDE UI Localization

In addition to CnWizards' own multi-language interface, CnPack IDE Wizards can localize part of the Delphi IDE UI for specific versions. This feature mainly translates some English menus and common UI text into Chinese.

Because Delphi versions differ in internal implementation, available coverage can vary by IDE version. Please verify the visible result in your target Delphi version after enabling it.

Requirements

  • Current CnWizards language is Chinese.

  • IDE UI localization is enabled in the Languages menu.

How to Use

  1. Open CnPack -> Languages in Delphi IDE.

  2. Enable the IDE UI localization option.

  3. If needed, restart IDE to refresh all windows and menus.

If some dialogs are still not localized in a specific IDE version, this is usually due to version compatibility differences. Upgrading to a newer CnWizards build is recommended for better coverage.
