歡迎使用 CnPack IDE 專家包

歡迎您使用 CnPack IDE 專家包 For Delphi/C++Builder/BDS。該專家包是針對 Delphi/C++Builder/BDS 開發的一組 IDE 功能擴展工具，用來提高程式師開發的效率。

專家包以 DLL 的方式集成到 IDE 中，包括一組以菜單和快捷鍵調用的工具、擴展的屬性編輯器、組件編輯器及代碼、窗體資源模板等。當前支持的開發環境有 Borland Delphi 5/6/7, C++Builder 5/6 和 BDS 2005/2006、RAD Studio 2007/2009/2010/XE/XE2/XE3/XE4/XE5/XE6/XE7/XE8/10 Seattle/10.1 Berlin/10.2 Tokyo/10.3 Rio/10.4 Sydney/11 Alexandria/12 Athens/13 Florence，其中部分工具僅在特定的開發環境中可用。

CnPack 專家包是開放源碼的自由軟體，您可以自由地在非商業應用中使用、複製該工具而不需支付任何費用，更多的說明請查看 CnPack IDE 專家包許可協議。

CnPack 專家包是 CnPack 開發包的一部分，由 CnPack 開發組 開發。該開發組由互聯網上熱愛 Delphi/C++Builder/BDS 程式開發的一群中國程式師自發組成，其目標是開發中國人自己的真正優秀的第三方免費開發包，更多的內容請查閱 關於 CnPack。

該專家包仍在持續開發中，建議您連接到 CnPack 開發網站 上查看最新的更新記錄。同時，專家包中內置的升級檢測功能可以在 IDE 啟動時自動聯網檢測最新的專家包升級，建議您儘快更新到最新的版本。在 歷史更新記錄 中，您可以查看到最新的功能及錯誤修正說明。

您在使用中有任何的問題或建議，請與我們聯繫：master@cnpack.org，開發組將盡力為您提供幫助！

　

感謝您對 CnPack 的支持！
