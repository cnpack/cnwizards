刪除注釋專家

該專家用於刪除單元檔中的注釋。運行該專家彈出的對話方塊如圖所示：

處理內容選擇

該部分可供用戶選擇要處理的物件：

  • 當前編輯的選擇區：刪除當前編輯區中的選中部分的注釋。

  • 當前編輯的單元：刪除當前編輯單元中的注釋。

  • 當前所有打開的單元：刪除當前所有打開單元中的注釋。

  • 當前工程中的所有單元：刪除當前工程中的所有單元中的注釋。

  • 當前工程組中的所有單元：刪除當前工程組中的所有單元中的注釋。

處理選項

該部分供用戶設置刪除注釋時的處理選項：

  • 刪除注釋塊的所有內容：處理注釋時，刪除被處理注釋的所有內容，包括注釋塊本身。

  • 只刪除注釋中的擴展 ASCII 字元：處理注釋時，只刪除注釋中的擴展 ASCII 字元，也就是刪除全形字符，注釋塊本身不被刪除。一般用於代碼移植到英文平臺時的處理。

  • 處理 Delphi 編輯指令：此選項只在 Delphi 中有效。選中此選項後，編譯指令也被當作注釋來進行處理，一般不推薦這種選擇，因為可能會影響到代碼編譯。

  • 處理 Todo List：選中此選項後，代碼中的 Todo List 也被當作注釋處理的物件來進行處理。

  • 處理項目源文件：選中此選項後，項目源文件中的註釋也將被處理。

  • 保留以內容中以下字串開頭的塊注釋：選中此選項後，代碼中的塊注釋（Delphi 中的{ } 和 BCB 中的 /* */ ）內容的開頭部分出現編輯框中的字串之一時，該注釋不會被進行處理。不同的字串以半形逗號進行分隔。舉例說明：如果保留編輯框輸入的內容是“*,(*,*),#”，則 Delphi 中，不會被處理的注釋包括{*...}、{(*...}、{*)...} 和 {#...} 四種形式（不支持 (*...*) 形式的注釋保留）。該項是為了方便用戶自定義保留需要的注釋。

  • 處理完畢後合併相鄰空行：大塊註釋處理完畢後，源碼中往往會留下大堆空白，選中此選項可將源碼中的連續空行合併為一行以減少空行數。
