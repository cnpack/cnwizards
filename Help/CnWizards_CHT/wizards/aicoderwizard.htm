AI 輔助編碼專家

該專家能夠連接並調用幾個常用的 AI 公開引擎之一，提供 IDE 內的 AI 輔助編碼功能。各 AI 引擎的 API 密鑰需用戶自行申請。使用前，請先運行該專家的配置，在對話框中填入申請的 API 密鑰方能使用，如圖所示：

AI 輔助編碼專家設置對話框

該對話框列出了可用 AI 引擎的參數，可供用戶選擇使用的 AI 引擎，以及設置每一個 AI 引擎的模型、API 密鑰等。

  • 當前 AI 引擎：選擇當前要使用的 AI 引擎，該引擎一般必須配置有效的 API 密鑰。

  • 超時秒數：網絡通訊的超時秒數設置，0 表示使用系統默認值。

  • 使用代理：如勾選，則代表指定連接該 AI 引擎所需的代理服務器，可以是 HTTP 或 SOCKS5 代理，格式如「192.168.1.1:1080」。暫不支持用戶名密碼。

  • 對話時發送歷史消息數：指定每次與 AI 引擎在聊天框內對話時攜帶最近幾條歷史記錄以記憶上下文。

  • 收藏最大條目數：指定 AI 聊天窗口內收藏夾能容納的最多條目數，0 表示無限制。

  • 編輯器中使用快捷鍵 (Ctrl +) Alt + Enter 續寫代碼：是否啟用兩個快捷鍵在編輯器中調用續寫代碼的功能，其中 Alt + Enter 是將 AI 撰寫的代碼直接輸入至編輯器當前光標後，Ctr + Alt + Enter 則是將結果輸出至 AI 對話窗口中供閱讀、複製。

  • 請求地址：AI 引擎的 API 在線請求地址，由該 AI 引擎提供。

  • 模型名稱：選擇該 AI 引擎提供的可用的模型名稱。

  • 在線獲取模型列表：以當前頁面參數在線調用該 AI 引擎提供商接口以在線獲取最新的模型列表，如成功則更新在下拉列表中。注意部分 AI 引擎提供商不支持該操作。

  • 採樣溫度：即 Temperature 參數，控制 AI 思維的發散程度。

  • API 密鑰：需填入在該 AI 引擎網站上註冊並申請的合法 API 密鑰。

  • 流式輸出：不勾選則控制讓 AI 引擎一次性返回完整應答數據，否則連續多次返回。

  • 申請：點擊申請鏈接可在線註冊並申請該 AI 引擎的 API 密鑰。

  • 重置：該按鈕可將當前 AI 引擎的設置恢復至默認狀態，但不清空 API 密鑰。

  • 快捷鍵：彈出快捷鍵設置對話框，設置菜單項的快捷鍵。

  • 確定：保存修改內容並關閉對話框。

  • 取消：取消修改並關閉對話框。

  • 幫助：顯示幫助信息。

 

AI 輔助編碼專家菜單

AI 輔助編碼專家的菜單項如圖所示：

  • 解釋代碼：向當前 AI 引擎發送請求，讓其解釋編輯器中選中的代碼塊，返回的內容將顯示在 AI 對話窗口中。

  • 檢視代碼：向當前 AI 引擎發送請求，讓其針對編輯器中選中的代碼塊進行檢視，也就是常說的 Code Review。返回的檢視意見將顯示在 AI 對話窗口中。

  • 生成測試用例：向當前 AI 引擎發送請求，讓其針對編輯器中選中的代碼塊生成一個無界面交互的測試用例。返回的結果將插入當前編輯器光標所在處，同時也顯示在 AI 對話窗口中。

  • 續寫代碼：向當前 AI 引擎發送請求，讓其針對編輯器中當前光標處的代碼或註釋續寫一段代碼，返回的結果將插入當前編輯器光標所在處。該功能默認熱鍵為 Alt+Enter。

  • AI 對話窗口：顯示 AI 對話窗口。

  • 設置：顯示上文的設置對話框。

 

AI 對話窗口

AI 對話窗口如圖所示。上部是聊天消息顯示區，下部是本地聊天消息發送區：

  • 切換發送框：顯示或隱藏發送消息的區域。

  • 清空消息：清空消息列表對話框。

  • 提問時附上編輯器內選中的代碼：消息區內發送消息時，是否附上編輯器內選中的代碼供 AI 分析。

  • 字體：修改消息框與發送框的字體。

  • 設置：顯示上文的設置對話框。

  • 幫助：顯示此幫助。

  • 收藏下拉菜單：顯示收藏下拉菜單，供快速在本地聊天消息發送區輸入收藏的內容。

  • 添加至收藏：將本地聊天消息發送區的內容添加至收藏菜單。

  • 清空收藏：刪除全部收藏菜單。

  • 發送：發送本地聊天消息區的內容給當前 AI 引擎。

 

右鍵點擊消息體將彈出上下文菜單：

  • 複製：複製選中的消息體內容。

  • 複製代碼：複製選中的消息體中被 ``` 括起來的內容當作代碼處理。

  • 添加到收藏：添加選中的消息體中的文字到收藏。
