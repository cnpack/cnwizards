IDE 專家基本操作

專家主菜單

IDE 專家包是緊密集成在 Delphi/C++Builder IDE 環境中的，用戶安裝專家包後，會在 IDE 主功能表上看到一個新的功能表項 CnPack，點擊該功能表項，顯示功能表列表如下圖所示：

其中，每一個功能表項對應著一個獨立的 IDE 專家，專家包當前仍在持續開發中，您看的功能表列表內容可能比上圖要多。

某些專家只有在特定條件下才可使用，如“元件選取”專家要求當前活躍視窗為表單設計器，如果用戶正在編輯源代碼，則該選項顯示變灰。

“Languages”子功能表下可以選擇專家介面語言，當前支援 簡體中文、繁體中文、英文 三種語言。

“幫助”子功能表下可以查看專家幫助、報告錯誤、訪問開發網站、顯示版權資訊和每日一帖等。

功能表上的所有快捷鍵，用戶都可以自己定制，如果希望禁用某些專家讓其不顯示在功能表中，可以在設置介面中設置，詳見 專家包設置。

　

工具欄按鈕

除了使用功能表項和快捷鍵調用外，您還可以將專家工具放置到 IDE 的工具欄中，設置方法如下：

方法一：右鍵點擊 IDE 工具欄，在彈出功能表中選擇“Customize...”。

方法二：選擇 IDE 主功能表下的“View”|“Toolbars”|“Customize...”。

彈出如下設置視窗：

“Commands”頁中列出了所有可用的命令，您可以將 CnPack IDE 工具直接拖到 IDE 工具欄上。
