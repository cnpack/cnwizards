代碼塊註釋反轉工具

該工具用於反轉當前選擇的源代碼塊的 // 風格的註釋，支持 Delphi/C++Builder，默認快捷鍵為 Ctrl+/。

如果選擇代碼的第一行是未註釋的，則自動為選擇的全部代碼行增加行首 // 註釋，如果第一行是已註釋的，則每一行根據是否註釋進行反轉。

使用前可以先選擇需要執行轉換的代碼行，如未選擇，則默認處理當前行。

其設置界面如圖：

  • 切換註釋後光標自動移至下一行：控制切換註釋後光標是否移動到當前行的下一行，可用於連續切換註釋的場合。默認關閉。

  • 註釋模式：控制切換註釋時的縮進模式。

    • 將 // 插入至行首：註釋代碼時將 // 插入至行首。

    • 將 // 插入至行首非空格處：註釋代碼時跳過行首空格，將 // 插入至行首第一個非空格字符。如果整行全空，則 // 插入在行首。

    • 使用 // 替換行首空格：註釋代碼時如果行首存在兩個或兩個以上的空格，則將 // 替換行首的兩個空格，不改變代碼本身的縮進。如無空格，則 // 插入在行首。
