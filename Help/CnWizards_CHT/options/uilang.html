IDE 界面漢化

除專家包自身多語言外，CnPack IDE 專家包還提供對部分 Delphi 版本 IDE 界面文本的漢化能力。該功能用於將 IDE 中部分英文菜單和界面文字替換為中文，便於中文用戶日常使用。

該功能主要覆蓋菜單與常見界面文本，因 Delphi 各版本差異較大，不同版本可覆蓋範圍可能不同。建議在啟用後結合您的 IDE 版本進行實際確認。

啟用條件

  • 當前專家包語言為中文。

  • 在 Languages 菜單中啟用 IDE 界面漢化功能。

使用說明

  1. 在 Delphi IDE 中打開 CnPack -> Languages。

  2. 啟用 IDE 界面漢化相關選項。

  3. 如有提示，請重啟 IDE 以確保全部界面刷新。

如果您在特定版本中發現個別界面仍為英文，屬於版本兼容差異的正常現象。可優先升級到較新版本的專家包以獲得更完整的適配。
