窗口置頂與折疊擴展專家

視窗置頂與折疊擴展專家能為 IDE 中的非模態視窗的標題欄增加置頂和折疊等按鈕（折疊按鈕的功能是將視窗縮小至標題欄大小以少佔用桌面空間），默認會增加三個按鈕：選項下拉功能表按鈕、折疊按鈕和置頂按鈕。下圖是 Object Inspector 增加標題欄按鈕後的效果：

  • 選項下拉功能表：點擊此按鈕後彈出有關於標題欄按鈕的下拉功能表。

  • 折疊按鈕：點擊此按鈕後，視窗會折疊縮小至標題欄大小，再點擊一次還原。部分表單可能因為本身設置了 AutoSize 或 Constraints 屬性導致不能縮小至標題欄大小，使用時請注意。

  • 置頂按鈕：點擊此按鈕後本視窗將置頂，再點擊一次還原。

  • 將此表單加入遮罩列表：本專家允許用戶設置根據表單類名來決定是否顯示標題欄按鈕的篩檢程式，本功能表項的功能是將本表單的類名加入過濾列表以阻止顯示標題欄按鈕。

  • 設置：點擊此按鈕後將彈出設置視窗。

本專家的設置視窗如下圖所示：

　

  • 按鈕類型：控制在標題欄中增加的按鈕類型，包括是否顯示置頂、折疊、設置按鈕等。

  • 忽略標題欄文字寬度：如不勾上此選項，按鈕將顯示在標題欄文字之後，如表單標題欄文字過長，按鈕則不顯示。勾上此選項後將忽略標題欄文字的長度，而總在右邊顯示標題欄按鈕。

  • 視窗折疊使用動畫效果：控制是否在視窗的折疊過程中使用動畫縮進展開效果。

  • 啟用自定義表單類名過濾以阻止部分視窗顯示擴展按鈕：部分非模態表單是不需要顯示置頂等按鈕的，如 IDE 主表單等，而且用戶也可能不需要某些類型的表單顯示其標題欄按鈕。這奡ㄗ悀F根據表單類名的過濾功能，凡是在列表中並且勾上的過濾規則都會生效，和規則中標識的表單類名相同的表單，都不會顯示標題欄按鈕。兩下拉清單的下拉清單媢w置了 IDE 中常見的窗口類名和其注釋，列表如下：

  • 替換：將兩個下拉清單中編輯的規則類名和注釋替換選中的規則。

  • 添加：將兩個下拉清單中編輯的規則類名和注釋作為一條新規則加入到規則列表中。

  • 刪除：刪除規則列表中的選中規則。

  • 確定：關閉此視窗，保存選項設置內容並更新生效。

  • 取消：關閉此視窗，忽略修改的選項內容。

  • 說明：顯示說明資訊。
