1. Introdução ao CnWizards
=====================================================

    O CnWizards é o produto principal do CnPack. Seu nome completo em 
inglês é 'CnPack IDE Wizards'. É um conjunto de ferramentas plug-in 
gratuito usado no Delphi/C++Builder/BDS/RAD Studio para aumentar a 
eficiência de desenvolvimento.

    O CnWizards foi projetado para auxiliar programadores que utilizam 
Delphi 5, 6, 7, C++Builder 5, 6 e BDS 2005, 2006, RAD Studio 2007, 2009, 
2010, XE, XE2, XE3, XE4, XE5, XE6, XE7, XE8, 10 Seattle, 10.1 Berlin, 
10.2 Tokyo, 10.3 Rio, 10.4 Sydney, 11 Alexandria, 12 Athens, 13 Florence.

    A versão oficial do CnWizards inclui os seguintes idiomas: 
Chinês Simplificado, Chinês Tradicional, Inglês, Russo, Alemão,
Francês e Português.

4. Estrutura
======================================================================

    Existem vários diretórios nos códigos-fonte do CnWizards:

    1. Bin. Contém os binários de saída e outros arquivos de tempo de 
            execução, incluindo arquivos de idioma.
    2. Dcu. Diretório temporário para compilação.
    3. Doc. Contém todos os documentos do CnWizards.
    4. Help. Contém os arquivos HTML de origem do arquivo de ajuda do 
            CnWizards.
    5. Install. Contém scripts de instalação.
    6. Make. Contém scripts de compilação automática.
    7. Source. Contém todos os códigos-fonte do CnWizards.
    8. Tool. Contém os códigos-fonte de outras ferramentas do CnWizards.

6. Compilação e Construção
======================================================================

    Primeiro você precisa baixar o código-fonte do cnpack e descompactá-lo 
no mesmo diretório onde o código-fonte do cnwizards está. Em seguida, abra 
o arquivo de projeto no diretório "Source" para compilar. O diretório 
"Tools" contém os códigos-fonte de outras ferramentas do CnWizards.

    A ferramenta de compilação automática está no diretório "Make". Você 
pode executar "want.exe" na janela de comando para obter mais detalhes.
